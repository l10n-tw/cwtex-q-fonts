目的

前人所發佈之 cwTeX 字型仍有許多不完美之處，例如字符的寬度或位置不統一。本計劃的主要目的是修正 cwTeX 字型，願能使字型達更高水準。

修正方針

五個字體皆由 cwTeX Type 1 字型（簡稱為 cwt1）Version 1.1（見 cwt1 1.1 釋出網頁）以 FontForge 組合而成。相較於 cwt1，已針對五種字體的中文常用標點符號做了調整，皆以

  • 貼近基線；
  • 左, 右括號或引號分別向右、左集中；以及
  • 將漢字與符號之字寬統一

為方針所進行。
